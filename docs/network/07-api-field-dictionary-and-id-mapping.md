# DGUT Ulearning 字段字典与 ID 映射说明

## 1. 文档范围

本文档用于统一解释网站中常见字段、对象层级、ID 映射关系及其语义差异。

## 2. 核心 ID 字段字典

| 字段名 | 层级 | 类型 | 样例值 | 说明 |
| --- | --- | --- | --- | --- |
| `courseId` | 课程层 | Integer | `55892` | UA 学习页课程 ID |
| `classId` | 班级层 | Integer | `944483` | 班级 ID |
| `chapterId` | 目录层 / 行为层 | Integer | `136397207` 或 `2581780` | 同名字段存在两种语义，需结合上下文区分 |
| `nodeid` | 内部章节层 | Integer | `2581780` | 章节内部节点 ID |
| `itemid` | 小节层 | Integer | `2584634` | 小节学习记录 ID |
| `pageid` | 页面层 | Integer | `6848124` 或 `2584635` | 在不同上下文里可能表示页面对象 ID 或页面记录 ID |
| `relationid` | 页面记录层 | Integer | `2584635` | 页面记录关联 ID |
| `resourceid` | 视频业务层 | Integer | `1717243` | 视频资源业务 ID，即 `videoId` |
| `resourceFullurl` | 资源文件层 | String | `https://obscloud.ulearning.cn/resources/web/17405354188653939.mp4` | 视频播放完整地址 |
| `media-id` | 资源文件层 | String | `17405354188653939` | MP4 文件主键 |

## 3. `chapterId` 语义区分

| 场景 | 字段值 | 实际含义 |
| --- | --- | --- |
| 课程入口 URL | `136397207` | 目录层章节 ID，对应 `Chapter.idForClass()` |
| `watchVideo` 请求体 | `2581780` | 章节内部节点 ID，对应 `Chapter.id()` |

### 3.1 结论

| 编号 | 结论 |
| --- | --- |
| 1 | 不能仅凭字段名判断含义 |
| 2 | 入口 URL 中的 `chapterId` 与 `watchVideo` 中的 `chapterId` 不属于同一层级 |
| 3 | 目录接口和章节详情接口联合后才能完成正确映射 |

## 7. `pageid` 字段的上下文差异

| 场景 | 字段名 | 样例值 | 实际语义 |
| --- | --- | --- | --- |
| 目录或页面详情对象 | `id` | `6848124` | 页面对象 ID |
| 学习记录同步对象 | `pageid` | `2584635` | 页面记录关联 ID，即 `relationid` |

### 7.1 结论

| 编号 | 结论 |
| --- | --- |
| 1 | 页面内容装配依赖页面对象 ID |
| 2 | 学习记录匹配依赖页面记录关联 ID |
| 3 | 不能直接将页面对象 ID 与学习记录 `pageid` 视为同一字段 |

| 场景 | 推荐称呼 |
| --- | --- |
| URL 里的 `chapterId` | `chapterIdForClass` 或“目录层章节 ID” |
